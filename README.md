# Tetris in Python

A simple, fully-playable implementation of **Tetris** written in **Python**.  
This project demonstrates game-loop programming, collision detection, piece rotation, line clearing, and rendering using a lightweight graphics library `pygame`.

---

## Features
- ✔️ Classic Tetris gameplay  
- ✔️ 7 tetromino types with rotation
- ✔️ Increasing difficulty
